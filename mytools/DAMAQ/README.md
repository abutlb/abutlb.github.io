# DAMAQ

### Build Better Data.

DAMAQ is an intelligent browser-based data quality engine designed for profiling, validation, cleaning, and DAMA-oriented quality assessment.

Originally built as a TDQ-based quality analyzer, DAMAQ was completely re-engineered into a modular data quality engine with advanced profiling, rule validation, audit tracking, and professional reporting capabilities.

# ✨ Features

## 📊 DAMA-Oriented Quality Assessment

* 7 DAMA quality dimensions
* Advanced quality scoring
* Dimension-based evaluation
* Enterprise-oriented quality analysis

---

## 🔍 Advanced Data Profiling

Analyze datasets with detailed column-level statistics:

* Mean
* Median
* Min / Max
* Null Analysis
* Entropy Analysis
* Frequency Distribution
* Duplicate Detection
* Data Type Detection

---

## ⚙️ Rule Engine

Create and apply custom data quality rules:

* Completeness Rules
* Validity Rules
* Consistency Rules
* Integrity Rules
* Timeliness Rules
* Uniqueness Rules

---

## 🧹 Smart Cleaning Engine

Built-in cleaning and normalization modules:

* Date normalization
* Text normalization
* Phone normalization
* Encoding fixes
* Duplicate merging
* Outlier handling
* Smart missing-value filling

---

## 📁 Professional Reporting

Generate structured multi-sheet reports:

* XLSX Export
* PDF Export
* HTML Reports
* Charts & Visual Insights

---

## 📜 Audit Logging

Track every cleaning and transformation operation for transparency and traceability.

---

## ⚡ Modern UX

* Interactive dashboard
* Skeleton loading
* Toast notifications
* Browser-based processing
* No backend required
